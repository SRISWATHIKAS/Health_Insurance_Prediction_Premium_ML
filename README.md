# Health_Insurance_Prediction_Premium_ML
# Objective
To predict the health insurance - charges based age, BMI, and smoking status, and then developing a predictive model using machine learning methods (specifically, linear regression). The model's performance is evaluated using metrics such as MSE and R-squared to determine the accuracy of the model.
# Installing and Importing Necessary Libraries

// Import all the necessary Python libraries for data manipulation, visualization, model building, and evaluation :
+ NumPy(Numerical Python) and Pandas(Panel Data) - for numerical and data manipulation.
+ Matplotlib and Seaborn - for data visualization.
+ Scikit-learn - for building and evaluating the ML models.
